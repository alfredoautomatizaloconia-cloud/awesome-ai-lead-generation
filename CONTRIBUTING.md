# Contributing to Awesome AI Lead Generation

Thank you for contributing! This list aims to be the definitive resource for AI-powered lead generation tools.

## Guidelines

### Adding a Tool

1. **One tool per PR** for easier review
2. **Alphabetical order** within each category
3. **Format**: `[Tool Name](URL) - Brief description. \`Language/Platform\` \`Pricing\``
4. **Verify** the link works and the tool is actively maintained
5. **Include** the pricing model: `Free`, `Freemium`, `Paid`, `SaaS`, `API`

### Quality Standards

- Tool must be **AI-powered** or directly useful for AI-driven lead generation
- Must have been **updated within the last 12 months** (for open-source tools)
- No affiliate links
- No tools that violate platform TOS or enable spam

### Categories

If your tool doesn't fit an existing category, suggest a new one in your PR description.

### Removing a Tool

If a tool is discontinued, no longer maintained, or has changed significantly, please open an issue or PR to remove it.

## Code of Conduct

Be respectful. This is a community resource. We prioritize quality over quantity.
